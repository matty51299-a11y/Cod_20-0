# 20-0: The All-Time CDL Gauntlet
## Pre-CDL 12-Team Database Patch V2

This is a replacement patch for the older/pre-CDL section of the database.

The previous V1 file did not follow the brief properly because several older titles only had around 6 teams. This V2 expands each older title to a 12-team historical pool where reliable Champs/major event data is available.

Use this as a proof-checkable balance draft, not as final gospel. Rosters are based on public historical event/team pages, but ratings are deliberately game-balance ratings for this draft simulator.

## Rating economy

- 96-97 = generational/MVP season, extremely rare
- 93-95 = elite superstar season
- 90-92 = top starter/star card
- 87-89 = good starter
- 84-86 = average or inconsistent starter
- 80-83 = weak starter/fringe card
- 76-79 = bad/emergency/open-bracket card

## Tier rules

- elite = should feel like a premium pull
- strong = good team, usually one or two strong choices
- mid = useful but flawed
- weak = mostly bad choices with maybe one usable player
- disaster = painful draft card
